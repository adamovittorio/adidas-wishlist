# adidas-wishlist

A simple wishlist

- [Get Started](#get-started)
- [Development](#development)

## Get Started

- Install node_modules

```bash
npm install
```

- Run test

```bash
npm test
# or
npm run coverage
```

- Run App (server + client)

```bash
npm start
```

## Development

- Start nodemon for watching server sourcecode changes and webpack for client

```bash
npm run start:dev
```

- Start test suite

```bash
npm run test:watch
```
